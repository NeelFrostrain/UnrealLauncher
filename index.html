Frequently Asked Questions

Find answers to common questions about Unreal Launcher.

How do I add an Unreal Engine installation?
  1. Open Unreal Launcher and navigate to the Engines tab
  2. Click the "Add Engine" button (usually a + icon)
  3. Browse to your Unreal Engine installation folder (e.g., C:\Program Files\Epic Games\UE_5.3)
  4. Select the folder and click Confirm
  5. The engine will be indexed and appear in your engines list
  6. You can now launch projects with this engine version
How do I add my Unreal Engine projects?
  1. Go to the Projects tab in Unreal Launcher
  2. Click the "Add Project" or "Import Projects" button
  3. Single project: Select a .uproject file directly
  4. Batch import: Select a folder containing multiple projects (up to 20 at a time)
  5. Wait for the projects to be indexed
  6. Your projects will now appear in your projects list, sortable by favorites or recent
How do I launch a project?
  1. Find your project in the Projects tab
  2. Click the Launch button on the project card
  3. Unreal Launcher will automatically open the project with the correct engine version
  4. The project will appear in your Recent Projects list
  5. Tip: Star your favorite projects for quick access from the Favorites tab
How do I use Unreal Launcher on Mac or Linux?

Mac and Linux are not officially supported yet, but you can build from source:

  1. Clone the repository: git clone https://github.com/NeelFrostrain/UnrealLauncher.git
  2. Install dependencies: npm install
  3. Start development: npm run dev
  4. Build for your OS: npm run build:mac or npm run build:linux
  5. The built application will be in the dist/ folder
  6. See the GitHub repository for detailed build instructions
What are the system requirements?
Minimum Requirements:
  • Windows 10 or Windows 11 (x64)
  • At least 2GB of RAM
  • An installed Unreal Engine version (5.0 or later recommended)

Recommended:

  • Windows 11
  • 8GB+ of RAM
  • SSD for faster project loading
  • Multiple Unreal Engine versions for flexibility
Why is Unreal Launcher so fast?

Unreal Launcher is built with Electron.js and React, which provide:

  • Lightweight codebase: Minimal dependencies keep the app fast
  • Efficient indexing: Projects are cached for instant access
  • Optimized rendering: Only visible content is rendered
  • Fast project discovery: Batch import with non-blocking operations
  • One-click launching: Direct engine-to-project connection
Can I use multiple Unreal Engine versions?

Yes! Unreal Launcher is designed to handle multiple engine versions:

  • Add as many engine installations as you like
  • Each project automatically uses its compatible engine version
  • Switch engine versions for a project anytime
  • Perfect for developers working on legacy and modern projects
  • Organize your workflow across UE 4.x, 5.x, and beyond
How do I update Unreal Launcher?
  1. Unreal Launcher has built-in auto-update via GitHub
  2. When an update is available, you'll see a notification
  3. Click Update to download and install the latest version
  4. The app will restart automatically
  5. Alternatively, download the latest from GitHub Releases
Is Unreal Launcher open source? Can I contribute?
